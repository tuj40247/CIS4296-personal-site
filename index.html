Gen Ed Discovery

I worked in a group of 5 to provide a web application to allow users to search and discovery information about the different Gen Ed Courses provided at Temple University. The application provided a simple and easy to use experience for users to browse the site and gain knowledge of Gen Ed courses of interest.

Course Registration Application

Created a web application that would allow users to register for available courses and allow them to choose their desired course delivery method of: Traditional, Online, or Hybrid. The application is small scale but can allow users to enter all the basic information needed and have the system display back to the user their information entered and their bill.
